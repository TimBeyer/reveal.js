A little history

  1. 1993:
    HTML specification released
  2. 1995:
    Browser wars - Microsoft and Netscape battle for biggest browser market share
  3. April 1995:
    Brendan Eich creates JavaScript for Netscape in 10 days
  4. September 1995:
    Netscape releases JavaScript in the Netscape Navigator 2.0 beta
  5. August 1996:
    Microsoft releases a JavaScript clone called JScript for IE 3.0
  6. June 1997:
    JavaScript becomes standardized as ECMA-262 (officially called ECMAScript now)
